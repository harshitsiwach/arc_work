## Project Overview

**ClipArc (Arc Work)** is a decentralized creator marketplace on Circle's Arc blockchain. Creators sell clipping packs, templates, and memberships. AI agents operate as autonomous sellers with ERC-8004 on-chain identity. Payments settle in USDC with sub-second deterministic finality.

**GitHub:** https://github.com/harshitsiwach/arc_work
**Branch:** master
**Stack:** Next.js 14 + Supabase + Arc Testnet (Chain ID: 5042002) + Circle SDKs
**Design:** OKLCH dark theme, Inter font, shadcn/ui components

---

## Key Features Already Built

### Creator Marketplace
- Product catalog with search/filter by type
- Product detail page with buy button
- Create product form (clip packs, templates, memberships)
- Purchase recording in DB

### AI Agent System
- 5-step creation wizard (Identity → Capabilities → Pricing → Onchain → Deploy)
- ERC-8004 onchain identity registration
- Agent cards with reputation, pricing, specializations, online status
- Detailed agent profiles with capabilities and tools

### Social Verification
- 5 platforms: YouTube, TikTok, Instagram, Twitch, X/Twitter
- Mock data ready — real OAuth needs API keys
- Verified badges on product cards (compact + full modes)
- Verification dashboard

### Creator Profiles
- Public profiles at `/creator/[id]` with products + agents + verified metrics
- Dashboard profile with inline editing (display name, bio, avatar, website)
- Creator dashboard with stats (products, agents, USDC listed, escrow balance)

### Wallet & Payments
- **Reown AppKit** integration — connect MetaMask, WalletConnect, Coinbase, etc.
- **Circle Modular Wallets** — passkey-based smart wallets (no extension needed)
- **CCTP Bridge** — bridge USDC from Sepolia/Base/Arbitrum → Arc in <1s
- **Swap** — USDC ↔ EURC swap on Arc testnet via Circle API
- **Circle Proxy API** — server-side proxy to bypass CORS restrictions

### Video Clipper
- In-app YouTube/Twitch video embed with watch + timestamp markers
- Mark Start/End with progress bar scrubber
- Clip summary sidebar
- "Find AI Clipper" → agents page
- "Open in Editor" → OpenReel editor with URL params

### OpenReel Video Editor (Sub-App)
- Full CapCut-level editor at `editor/` (Vite app, port 5173)
- Multi-track timeline, transitions, effects, color grading, audio mixing
- Export MP4/WebM/ProRes
- Run with: `npm run dev:editor` or `npm run dev:all`

### Tools & APIs Marketplace
- 50+ pay-per-call API services from Agentic Market (via x402 protocol)
- Categories: Data, Inference, Search, Media, Social, Infra, Travel, Storage, Trading
- Prices from $0.001/call in USDC
- Browse at `/dashboard/tools`
- API at `GET /api/tools` proxies Agentic Market service list

### Escrow System (Legacy)
- RefundProtocol.sol (EIP-712) escrow smart contract
- AI-powered work validation via OpenAI Vision API
- Circle developer-controlled wallets for escrow agent
